## State Transitions: BlogPost

```
DRAFT ──[Publish]──→ PUBLISHED
PUBLISHED ──[Unpublish]──→ DRAFT
DRAFT/PUBLISHED ──[Soft Delete]──→ DELETED (deletedAt set)
DELETED ──[Restore]──→ DRAFT
```
