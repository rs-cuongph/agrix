# Quick Start: Group Sidebar Menus

## Overview

Deploying and interacting with the grouped sidebar menus requires no special setup. The changes are confined to the `AdminSidebar` Next.js React component.

## How to Test

1. Ensure the web application is running locally (e.g., using `npm run web:dev`).
2. Navigate to `http://localhost:3000/admin`.
3. Observe the left-hand sidebar which will now display top-level group headings (Cửa hàng, Nội dung / Web, Hệ thống).
4. Click on a group heading to verify it expands into the respective child menus.
5. Click on a child menu and verify the group remains open while navigating across the corresponding admin pages.
